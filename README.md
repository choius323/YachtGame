# Yacht Game

YacthGame은 닌텐도의 51 Worldwide Games를 통해 유명해진 보드 게임의 한 종류이며, 안드로이드용으로 개발되었습니다.

## 기능

- 주사위를 굴리고 예상 점수를 표시
- 굴리지 않을 주사위 선택
- 점수 기록 및 확인
